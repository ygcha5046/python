# 파이썬 함수 정리

### **1. 문자열 다루기 문제**

| 함수 | 설명 | 예시 |
| --- | --- | --- |
| `len(s)` | 문자열 길이 | `len("hello") → 5` |
| `s.upper()` | 대문자 변환 | `"abc".upper() → "ABC"` |
| `s.lower()` | 소문자 변환 | `"ABC".lower() → "abc"` |
| `s.isalpha()` | 알파벳만인지 확인 | `"abc".isalpha() → True` |
| `s.isdigit()` | 숫자인지 확인 | `"123".isdigit() → True` |
| `s.split()` | 공백 기준 나누기 | `"a b c".split() → ['a','b','c']` |
| `s.replace(a, b)` | 문자 교체 | `"aabb".replace("a", "x") → "xxbb"` |
| `s.strip()` | 양쪽 공백 제거 | `" hi ".strip() → "hi"` |
| `s[::-1]` | 문자열 뒤집기 | `"abc"[::-1] → "cba"` |
| `'구분자'.join(리스트)` | 리스트 합쳐 문자열 만들기 | `' '.join(['a','b']) → 'a b'` |

### **2. 리스트/배열 문제**

| 함수 | 설명 | 예시 |
| --- | --- | --- |
| `len(arr)` | 길이 | `len([1,2,3]) → 3` |
| `sum(arr)` | 합계 | `sum([1,2,3]) → 6` |
| `max(arr)` | 최댓값 | `max([1,5,3]) → 5` |
| `min(arr)` | 최솟값 | `min([1,5,3]) → 1` |
| `arr.append(x)` | 끝에 추가 | `[1,2].append(3) → [1,2,3]` |
| `arr.pop()` | 마지막 제거 | `[1,2,3].pop() → 3` |
| `arr.sort()` | 오름차순 정렬 | `[3,1,2].sort() → [1,2,3]` |
| `arr.sort(reverse=True)` | 내림차순 정렬 | `[3,1,2] → [3,2,1]` |
| `x in arr` | 값 포함 여부 확인 | `2 in [1,2,3] → True` |

### **3. 조건문/반복문 문제 (제어문)**

| 문법/함수 | 설명 | 예시 |
| --- | --- | --- |
| `range(n)` | 0~n-1 반복 | `range(5) → 0~4` |
| `range(a,b)` | a~b-1 | `range(2,6) → 2~5` |
| `range(a,b,c)` | 간격 c | `range(10,0,-1) → 10~1` |
| `enumerate(arr)` | 인덱스+값 함께 반복 | `for i, v in enumerate(arr)` |
| `zip(a, b)` | 두 리스트 동시 반복 | `for x, y in zip(list1, list2)` |
| `if x in arr:` | 포함 여부 | `if 3 in [1,2,3]` |
| `continue` | 다음 반복으로 넘어감 | `if x % 2 == 0: continue` |

### **4. 수학/계산 문제**

| 함수 | 설명 | 예시 |
| --- | --- | --- |
| `abs(x)` | 절댓값 | `abs(-5) → 5` |
| `round(x, n)` | 반올림 (소수 n자리) | `round(3.1415, 2) → 3.14` |
| `int(x)` | 정수형으로 변환 | `int("10") → 10` |
| `float(x)` | 실수형으로 변환 | `float("3.14") → 3.14` |
| `str(x)` | 문자열로 변환 | `str(100) → "100"` |
